# CONSOLEAPP(APPDEV)

**Simple Task Manager Application**

Created by: **Gabriel Bolante**  
Course: **BSCS 3-2**

## Description
A C# console application that allows users to manage their daily tasks with features including:
- Add new tasks
- View all tasks
- Remove completed tasks
- Simple and intuitive menu interface

## Project Purpose
This project demonstrates the use of Git version control and branching strategies, including:
- Creating and managing branches
- Committing changes
- Pushing to remote repository
- Creating pull requests
- Merging branches
- Reverting commits

## Branches
- **main**: Stable version of the application
- **experimental/new-features**: Development branch with enhanced features (task counter, statistics)
